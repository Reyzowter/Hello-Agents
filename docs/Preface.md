# Preface

## Why We Built Hello-Agents

The year 2025 marked a decisive turning point in AI. The race to build larger foundation models was giving way to something more interesting: **building systems that can act, reason, and collaborate**. Agents.

Yet despite the explosion of interest, the learning resources available were either too shallow (API tutorials that skip the fundamentals) or too academic (papers that skip the implementation). We wanted something in between — a resource that treats you as an intelligent adult, explains the *why* behind every design choice, and gives you code you can actually run.

That's Hello-Agents.

## What This Book Is

Hello-Agents is a **systematic, practical tutorial** for building AI agent systems from scratch. It covers:

- The theoretical foundations of agent design (what an agent *is*, how it *reasons*)
- Hands-on implementation of classic paradigms (ReAct, Plan-and-Solve, Reflection)
- Popular frameworks (AutoGen, LangGraph, AgentScope) in real scenarios
- Advanced techniques: memory, context engineering, multi-agent protocols
- Complete real-world projects from start to finish

By the end, you won't just be a *user* of agent tools — you'll understand them deeply enough to build your own.

## Who This Is For

This tutorial is designed for:

- **Developers** with basic Python skills who want to build AI-native applications
- **Engineers** transitioning into AI product development
- **Students and researchers** exploring the LLM agent landscape
- **Job seekers** preparing for AI engineer interviews

You do *not* need a deep background in machine learning. You *do* need to be comfortable writing Python and calling APIs.

## How to Use This Book

The curriculum is structured in five parts, each building on the last:

1. **Part 1 (Chapters 1–3):** Understand what agents are and how LLMs power them
2. **Part 2 (Chapters 4–7):** Build agents by hand — no frameworks yet
3. **Part 3 (Chapters 8–12):** Add memory, context, protocols, training, and evaluation
4. **Part 4 (Chapters 13–15):** Apply everything to three complete real-world projects
5. **Part 5 (Chapter 16):** Build your own multi-agent application from scratch

**Recommended approach:** Read each chapter, then run every code example before moving on. The code is in the `/code` folder, organized by chapter.

If you're short on time, Parts 1 and 2 give you a solid standalone foundation. Come back for Parts 3–5 when you're ready to go deeper.
